Reactor Physics & Computational Analysis Laboratory · KINGS, Ulsan

We write the code that designs the core.

RPL builds its own reactor-physics code suite from the kernel up — Monte Carlo transport, pin-by-pin diffusion and real-time nodal solvers — and uses it to design next-generation cores: soluble-boron-free SMRs, sodium fast reactors and molten-salt systems.
